# Colonia Aero Club

Sitio web oficial del **Colonia Aero Club**, institución aeronáutica fundada el **31 de enero de 1941** en el departamento de Colonia, Uruguay.

La página fue desarrollada en HTML y CSS puro, con diseño adaptable para dispositivos móviles y enfoque institucional.

---

## ✈️ Contenido del sitio

El sitio incluye:

- Inicio institucional
- Sección “Quiénes Somos”
- Misión, Visión y Futuro
- Información del CIAC 016 DINACIA
- Galería fotográfica
- Contacto institucional
- Enlaces a Instagram y WhatsApp
